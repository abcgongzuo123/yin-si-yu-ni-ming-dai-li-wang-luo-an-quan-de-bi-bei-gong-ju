## 什么是代理？

代理是您的设备与您访问的网站之间的中介，它隐藏了您的真实IP地址。也就是说，网站只会看到代理的IP地址，而不是您的真实地址，从而增强了隐私和匿名性。代理有多种类型，例如住宅代理，它们看起来像普通用户的流量，因此不容易被检测到，是保护隐私的理想选择。
